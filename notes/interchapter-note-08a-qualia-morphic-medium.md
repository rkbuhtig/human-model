## 0. 이 노트가 필요한 이유

Chapter 08은 `Π_view`의 image scaffold를 기억 Graph에 접근하는 주소로 읽었다. 당시 연구 후기는 `image`를 시각 이미지로 좁게 읽을 경우 언어·절차·운동·정서·몸·비자발적 회상 경로를 막을 수 있다고 비판했다.

사용자의 후속 설명은 이 해석의 전제를 교정한다.

> **[USER DIRECT] 퀄리아는 언어를 쓰기 전부터 동물이 감각을 생존에 사용할 수 있게 하는 수단이다. 퀄리아는 슬라임처럼 계속 변형되며, 처음 보는 대상을 그에 맞는 형상으로 주물러 내재화한다. 언어는 이 유동적 형상을 고정하고 다시 접근하기 쉽게 만드는 후발 지식 도구다. 생각 자체도 경험이므로 고차 사고는 퀄리아를 다시 정교화한다.**

따라서 다음 순서는 틀렸다.

```text
language
→ qualia
→ access
```

현행 가설의 순서는 이쪽이다.

```text
sensation / body / memory / imagination
→ deformable QualiaMedium
→ present QualiaMorph
→ felt representation / action possibility

language
→ later anchor / mold / recombination handle
```

이 정정은 0120 원문의 역사적 의미를 바꾸지는 않는다. `image scaffold`가 어떤 정의역을 가질 수 있는지에 대한 현재 Bridge를 바꾼다.

## 2. 첫 교정 — 퀄리아는 언어의 산출물이 아니다

### 2.1 전언어적 생존 인터페이스

`[USER DIRECT]` 퀄리아는 인간 언어 이전에 존재한다. 동물은 이름을 모르더라도 빛·소리·냄새·맛·통증·균형·몸 내부 상태를 하나의 생존 가능한 장면으로 사용해야 한다.

```text
heterogeneous sensory streams
→ felt, integrated, actionable form
```

퀄리아가 하는 일은 원시 신호에 장식적 색을 붙이는 것이 아니다. 서로 다른 감각과 몸 상태를 현재 행동에 사용할 수 있는 형상으로 만든다.

### 2.2 ‘이미지’는 시각상이 아니다

이 모델에서 `image`는 사진 같은 시각상에 한정되지 않는다.

- visual shape
- sound·inner tone
- smell·taste gestalt
- pain·tension·temperature
- balance·orientation
- interoceptive pressure
- motor readiness
- temporal rhythm
- affective color
- spatial·relational form

을 포함하는 **느껴지는 다감각적 형상**이다.

따라서 후속 용어는 `ImageScaffold`보다 `PhenomenalScaffold` 또는 `QualiaMorph`가 안전하다.

### 2.3 생존과 정확성

생존에 적합한 형상이 곧 세계의 완전한 복사는 아니다.

```text
viable action-form
≠ exhaustive world model
≠ public evidence
≠ infallible truth
```

퀄리아는 유한한 생명이 세계 전체를 복제하지 않고도 접촉·회피·탐색·학습하게 하는 압축·형상화 매질 후보다.

## 3. 슬라임 비유를 형식 부품으로 바꾸기

### 3.1 QualiaMedium과 QualiaMorph

`[SYNTHESIS]` 하나의 `Qualia`라는 말에 매질과 현재 형상을 함께 넣으면 혼동이 생긴다.

```text
QualiaMedium_t
= 계속 변형될 수 있는 현상적 매질과 그 현재 물성

QualiaMorph_t
= tick t에서 그 매질이 실제로 취한 느껴지는 형상
```

매질은 빈 물질이 아니다. 과거 경험이 남긴 attractor·경사·점성·탄성·흉터를 이미 가진다. 현재 morph는 그 매질이 새 forcing 아래 취한 한 구성이다.

### 3.2 매질을 쓰는 별도 난쟁이는 없다

‘내가 퀄리아를 도구로 쓴다’는 표현을 문자 그대로 읽으면 안에 또 다른 관찰자를 요구한다.

```text
inner observer
→ reads qualia
→ who reads the observer?
```

`[BRIDGE]` 이를 피하려면 퀄리아 형상이 생기는 사건 자체가 현재의 느껴짐·생각·행동 편향이어야 한다.

> QualiaMorph를 보는 별도 자아가 있는 것이 아니라, QualiaMedium이 현재 형태를 취하는 사건이 곧 현재 체험의 일부다.

### 3.3 형상은 내용과 행동 가능성을 함께 가진다

새 사물에 대한 morph에는 다음이 함께 있을 수 있다.

```text
what-it-feels-like
what-it-resembles
what-can-be-done
what-matters-now
approach / avoid / inspect tendency
known / unknown boundary
```

따라서 QualiaMorph는 정적인 표상보다 **affordance-bearing phenomenal form**에 가깝다.

## 4. 경험은 외부 접촉만이 아니다

### 4.1 경험 source의 분리

`[USER DIRECT]` 생각하는 행위도 경험이다. QualiaMedium은 외부 현실뿐 아니라 회상·상상·사고실험을 실제 현재 형태로 겪는다.

```text
ExternalContact
BodyInteroception
Recollection
InternalSimulation
Dream
ReceivedStory
LanguageEvocation
```

모두 현재 QualiaMorph를 만들 수 있다.

### 4.2 경험의 현상적 실재와 외부 사실성

각 source가 만든 체험은 당사자에게 실제로 일어난 현재 사건일 수 있다. 그러나 source가 다르면 외부 주장에 대한 지위도 다르다.

```text
PhenomenalExperienceOccurred
≠ ExternalEventOccurred
≠ ExternalClaimSupported
```

내부 상상이 나를 바꿀 수 있다는 사실은 상상 내용이 세계에서 발생했다는 근거가 아니다.

### 4.3 source provenance의 필요

외부 접촉과 내부 simulation이 모두 비슷한 생생함을 만들 수 있다면, morph의 강도만으로 기원을 알 수 없다.

```text
Vividness
≠ Source
≠ Evidence
```

`[BRIDGE]` ArchiveTrace에는 내용뿐 아니라 최소한 다음 provenance가 필요하다.

```text
ExternalForcing
InternalSimulation
RecollectionOfPriorTrace
DreamGeneration
ReceivedTestimony
Inference
UnknownSource
```

source tag는 blame이나 truth token이 아니라 재검증 경로를 고르는 표지여야 한다.

## 5. 생각은 경험이다 — 가역성과 무흔적성을 분리하기

### 5.1 재귀적 자기 변형

생각은 QualiaMedium이 현재 형상을 만들고, 그 형상이 다시 다음 매질 상태를 바꾸는 loop다.

```text
QualiaMedium_t
→ Thought / Imagination
→ QualiaMorph_t
→ internal experience
→ PlasticUpdate
→ QualiaMedium_t+1
```

상상·연습·반추·내적 대화가 이후 어떤 형상을 더 쉽게 만드는 이유를 이 loop가 설명할 수 있다.

### 5.2 Ghost sandbox의 새 봉인

Ghost의 가역 sandbox는 외부 원장·행동·Narrative에 자동 commit되지 않는다는 뜻이다. 내부 물성이 한 치도 변하지 않는다는 뜻이 아니다.

```text
Reversible
≠ Traceless
```

극단적 사고실험도 Ghost 안에서 허용될 수 있다. 그 사고가 떠올랐다는 사실, 반복되며 salience를 바꿨다는 사실, 그것을 행동·서사로 인수했다는 사실은 서로 다르다.

```text
SpontaneousCandidate
≠ PlasticTrace
≠ NarrativeAdoption
≠ Intention
≠ PerformedAction
≠ ExternalFact
≠ Responsibility
```

### 5.3 새로운 trace type

0120의 `Receipt→σ→Bill`만으로 모든 thought-induced change를 받으면 정상 사고까지 부채가 된다.

```text
QualiaPlasticityTrace
≠ Scar
≠ Debt
≠ Obligation
```

`[BRIDGE]` 최소한 다음을 가를 필요가 있다.

| 변화 | 뜻 |
|---|---|
| `PlasticTrace` | 다음 morph의 확률·형상을 바꾸는 비권위 흔적 |
| `LearningUpdate` | 반복 가능한 구분·기술·접근 개선 |
| `ScarUpdate` | 고비용·고결합·경로 의존 변화 |
| `NarrativeAdoption` | 현재 자기가 의미·관계 지위를 인수 |
| `NormativeDebt` | 별도 규칙으로 발생한 의무 |

## 6. 언어 — 퀄리아의 기원이 아니라 결정화 도구

### 6.1 LanguageAnchor

`[USER DIRECT]` 언어는 이미 형성된 유동적 지식 형태를 고정해 접근하기 쉽게 한다.

```text
QualiaMorph
→ attach public symbol / name
→ LanguageAnchor
→ easier re-entry, composition, comparison, teaching
```

언어가 고정하는 것은 체험 전체의 동일 복사본이 아니라 반복 접근할 수 있는 손잡이·몰드·분할선이다.

### 6.2 같은 단어, 다른 퀄리아

```text
same token w
+ person A's body/history/terrain
→ QualiaMorph_A(w)

same token w
+ person B's body/history/terrain
→ QualiaMorph_B(w)
```

따라서:

```text
SameWord
≠ SameQualiaMorph
≠ SameMemoryNeighborhood
≠ SameStake
≠ SameNarrativeMeaning
```

`가족`, `사랑`, `안전`, `책임` 같은 단어는 사전 정의를 공유해도 전혀 다른 몸 감각·장면·비용·행동 경사를 열 수 있다.

### 6.3 의사소통은 퀄리아 전송이 아니다

말은 타인의 매질에 내 morph를 그대로 복사하지 않는다. 공통 symbol로 타인의 QualiaMedium이 어떤 관련 형태를 만들도록 자극한다.

```text
communication
≠ private qualia transfer

communication
≈ constrained cross-person re-morphing
```

성공적인 이해는 내부 체험의 완전한 동일성보다 필요한 구분·예측·행동·관계 조정이 충분히 맞는 상태일 수 있다.

### 6.4 언어의 이중 효과

언어는 형상을 보존하고 조합하게 하지만 새 입력을 너무 빨리 기존 이름 아래로 끌어들일 수도 있다.

```text
language as anchor
→ stable knowledge

language as rigid mold
→ premature closure / reification
```

이름을 안다는 사실이 대상을 새롭게 느꼈다는 사실을 대신하지 못한다.

## 7. 유동지능·결정지능·창의성

### 7.1 같은 매질의 다른 상태

`[USER DIRECT]` 처음 보는 사물에 맞는 형태를 만드는 능력은 유동지능과 연결된다. 이미 안정된 형상·언어·기술은 결정지능과 연결된다.

```text
FluidIntelligence
= novel forcing에 맞춰 QualiaMedium을 새 형태로 변형하는 능력

CrystallizedIntelligence
= 유효했던 morph·anchor·접근 경로가 안정 attractor가 된 상태
```

### 7.2 전문가와 초심자

```text
expert terrain
= deep, fine-grained attractors
→ fast discrimination, efficient action
→ novel input의 premature capture 위험

novice terrain
= shallow, sparse attractors
→ ambiguity를 오래 유지, wider exploration
→ 낮은 정확도·제약 감지·재현성 위험
```

전문가가 항상 덜 창의적이고 초심자가 항상 더 창의적이라는 정리는 아니다. 문제는 지식량보다 기존 attractor의 깊이를 상황에 맞게 조절할 수 있는가다.

### 7.3 창의성은 상전이 제어다

`[SYNTHESIS]` 창의성은 최대 유동성이나 최대 고정성이 아니라 둘 사이를 왕복하는 능력일 수 있다.

```text
soften existing attractor
→ generate multiple morphs
→ contact world / compare constraints
→ selectively crystallize
→ preserve future re-formability
```

이를 `QualiaAnnealing`이라 부른다.

```text
CreativeIntelligence
≈ Fluidity
 + useful crystallization
 + phase-transition control
```

### 7.4 beginner's mind

초심자의 마음은 지식 삭제가 아니다. 기존 지식이 새 forcing을 너무 빨리 점유하지 못하도록 attractor depth를 잠시 낮추는 meta-control에 가깝다.

창의적인 전문가는 형상이 없는 사람이 아니라 깊은 형상을 필요할 때 다시 유동화할 수 있는 사람이다.

## 8. 퀄리아의 유변학 — 무엇이 지속되는가

슬라임 비유를 끝까지 밀면 현재 형태만이 아니라 매질의 물성을 모델링해야 한다.

| 속성 | 질문 |
|---|---|
| Plasticity | 경험 뒤 변화가 얼마나 남는가 |
| Elasticity | forcing이 사라진 뒤 얼마나 돌아오는가 |
| Viscosity | 형태 변화에 얼마나 저항하는가 |
| Attractor depth | 기존 형상으로 얼마나 빨리 수렴하는가 |
| Permeability | 새 입력이 기존 지형에 얼마나 침투하는가 |
| Resolution | 얼마나 미세한 차이를 별도 형상으로 만드는가 |
| Hysteresis | 같은 입력에도 과거 경로 때문에 다른 형상이 나는가 |
| Compartmentalization | 영역 사이가 얼마나 섞이거나 차단되는가 |

### 8.1 Expertise

전문성은 특정 영역의 resolution과 attractor depth를 키운다. 장점은 빠르고 세밀한 형상화이고, 위험은 낯선 입력을 기존 몰드로 과도하게 끌어들이는 것이다.

### 8.2 Scar

Scar는 고통스러운 내용 하나가 아니라 매질의 변형 법칙 변화일 수 있다.

```text
lower threshold for some morphs
higher AfterCost
slower relaxation
rigid avoidance basin
context-specific compartment
```

### 8.3 Play·dream·safe exploration

`[BRIDGE]` 놀이·꿈·예술·사고실험은 외부 commit 압력을 낮추고 굳은 attractor를 다시 시험하는 annealing mode일 수 있다.

이것이 실제 수면·꿈 이론에 직접 승계되는지는 0121 이후 자료를 확인해야 한다 `[OPEN]`.

## 9. 기억 — 형상을 저장하는가, 다시 만드는 법을 저장하는가

### 9.1 저장 이미지 가설의 한계

QualiaMorph가 매 순간 몸·맥락·현재 매질의 물성에 따라 달라진다면 과거 morph를 그대로 저장·재생한다는 모델은 약하다.

### 9.2 DeformationRecipe

`[SYNTHESIS]` 기억은 완성된 퀄리아 이미지보다 비슷한 morph를 다시 만들 수 있는 제약·주소·변형 recipe를 저장할 수 있다.

```text
MemoryTrace ≈
  cue sensitivity
  + deformation tendency
  + linked neighborhoods
  + body/affect constraints
  + source/provenance
  + prior action/outcome relation
```

```text
Memory
≠ stored QualiaMorph

Memory
≈ constraint for reconstructing a QualiaMorph
```

### 9.3 Rehydration의 재해석

0120의 Rehydration은 압축 packet을 현재 Ψ 안에서 다시 펼치는 과정이었다. 현행 Bridge에서는:

```text
ArchiveTrace / DeformationRecipe
+ present QualiaMedium
+ current body/context
→ reconstructed QualiaMorph
```

로 읽을 수 있다.

같은 trace도 현재 매질이 달라졌다면 다른 체감·장면·의미로 돌아올 수 있다.

### 9.4 원본·재구성·사실

```text
same trace
≠ same QualiaMorph

vivid reconstruction
≠ accurate source attribution

current meaning
≠ past event content
```

따라서 기억에는 최소 두 lane이 필요하다.

```text
HistoricalOccurrence / provenance lane
PhenomenalReconstruction / current meaning lane
```

한 lane이 다른 lane을 자동 수정하거나 인증하지 못한다.

## 10. Ghost — 퀄리아 매질에서 후보가 사는 현재 공간

### 10.1 narrow Ghost와 broad Ghost-space

사용자의 Ghost 설명을 퀄리아 모델과 연결하면:

```text
narrow Ghost
= raw forcing에 반응해 툭 튀어나온 spontaneous candidates

broad Ghost-space
= candidates가 QualiaMorph로 체험되고
   비교·상상·전략 선택되는 현재형 자아 공간
```

Ghost는 QualiaMedium 전체와 같지 않다. 매질이 현재 만드는 후보·장면·전략 공간에 가깝다.

### 10.2 Body의 자리

```text
Body
→ sensory/interoceptive forcing
→ constrains possible morphs
→ authorizes or vetoes execution
```

몸은 Ghost의 단순 내용 항이 아니라 매질을 조건 짓고 행동을 실제 수행하는 별도 actor·boundary다.

### 10.3 Editor의 자리

```text
Ghost candidates
+ current QualiaMorph
+ Narrative terrain
+ projected irreversible traces
→ Editor deliberation
→ selected branch / hold
```

Editor는 자유롭게 shape를 발명하는 주권자가 아니다. 이미 형성된 attractor·몸의 조건·과거 Episode의 경사 아래에서 어떤 후보를 외부화할지 편성한다.

Editor가 broad Ghost-space의 mode인지 별도 sub-runtime인지는 `[OPEN]`이다.

## 11. TAD — 퀄리아 형상의 현실 승격을 제한하는 외골격

### 11.1 생성과 권한의 분리

```text
QualiaMedium
→ free morphing
→ Ghost candidate generation
→ thought experience
```

이 자유는 외부 사실과 공적 권한을 자동 생성하지 않는다.

```text
QualiaMorph
≠ Evidence
≠ Warrant
≠ PerformedAction
≠ AppliedFact
```

### 11.2 TAD의 자리

`[SYNTHESIS]` 현행 TAD는 인간 모델의 발생 중심이라기보다, 가소적인 내부 형상이 자기 강렬함만으로 현실 권한을 탈취하지 못하도록 하는 보호 문법이다.

```text
Qualia theory
= forms arise and change

Ghost / Editor theory
= forms are explored and selected

TAD
= selected/internal forms acquire or fail to acquire
  evidence, authority, event, application status
```

### 11.3 새로운 하위 비동일식

```text
InternalExperienceCanChangeMe
≠ InternalExperienceProvesWorld
```

이 구분은 `Influence ≠ Warrant`보다 인간 내부 쪽에 놓인 선행 문법이다.

## 12. 환각·자기기만 — 내용보다 source와 승격의 혼동

### 12.1 동일한 매질, 다른 forcing source

외부 접촉과 내부 simulation은 같은 QualiaMedium을 변형한다. 결과 morph가 비슷하게 생생할 수 있다.

```text

InternalSimulation → vivid morph
```

### 12.2 두 종류의 오류

```text
SourceConfusion
= internally generated morph를 external contact로 잘못 귀속

AuthorityElevation
= private morph를 public evidence/warrant로 승격 시도
```

둘은 연결될 수 있지만 같은 오류는 아니다.

### 12.3 반복 사고와 확신

반복은 attractor depth와 접근 용이성을 키울 수 있다.

```text
rehearsal
→ easier reconstruction
→ familiarity / vividness
```

그러나:

```text
Familiarity
≠ ExternalSource
≠ Truth
```

이 봉인이 생각을 경험으로 인정하면서도 현실 근거와 분리한다.

## 13. 사랑·관계·확장된 자기 경계

### 13.1 타자는 Qualia terrain을 바꿀 수 있다

반복된 돌봄·갈등·공동 시간은 특정 사람과 관련된 cue가 매질 전체를 특정 형태로 만들게 할 수 있다.

```text
other person's cue/state
→ strong QualiaMorph
→ body/action/access reconfiguration
```

그 사람의 존속과 손상이 내 유지 함수에 들어오면 관계는 단순 memory node를 넘어선다.

### 13.2 내재화와 소유권

```text
the other is deeply internalized
≠ the other is my property
≠ I may decide for the other
```

Qualia coupling은 Belonging·Stake·Responsibility·Identity를 바꿀 수 있지만 ValidAuthority를 자체 발행하지 않는다.

### 13.3 실제 경험의 인력

관계 축은 현재 의지가 자유롭게 고르는 knob가 아니다.

```text
care / betrayal / repair / promises / shared time
→ repeated morphs
→ slow terrain
→ current relational gradients
```

현재 Editor는 이 경사를 읽고 따르거나 비용을 치르며 거스를 수 있을 뿐, 과거 형성사를 0으로 만들 수 없다.

## 14. 생명 — 흐름을 내부에서 먼저 형태화하는 존재

### 14.1 형태 유지와 가소성

사용자의 큰 가설은 생명을 우주의 변화 흐름 속에서 흩어지지 않고 일부를 결합해 형태를 유지하는 존재로 본다.

퀄리아는 동물적 생명에서 외부 변화가 곧바로 자기 전체를 다시 쓰지 않게 하는 현상적 완충층 후보가 된다.

```text
world flow
→ body/sensory forcing
→ reversible Qualia morphing
→ candidate action / internalization test
→ selective stabilization
```

### 14.2 벽보다 변환기

```text
self boundary
≠ perfect isolation wall

self boundary
≈ selective transformation / repair / integration process
```

생명은 흐름을 차단하지 않고 통과시키면서도 자기 viability에 맞는 형태로 다시 결합한다.

### 14.3 범위 제한

`[CAUTION]` 이 노트는 퀄리아가 모든 생명에 존재한다고 확정하지 않는다. 사용자의 현재 범위는 적어도 감각을 생존에 사용하는 동물적 생명이다.

대사·항상성·수리·번식·몸–환경 경계 없이 QualiaMedium만으로 생명 전체를 설명할 수 없다.

## 15. 자아 — 같은 모양보다 이어지는 변형 문법

### 15.1 현재 형상은 계속 바뀐다

만약 퀄리아가 정적이지 않다면 동일한 QualiaMorph의 지속을 자아 동일성으로 삼을 수 없다.

### 15.2 Rheological identity

`[BRIDGE]` 자아의 일부는 무엇을 느끼는가뿐 아니라 **어떻게 변형되고 어떻게 다시 안정되는가**에 있을 수 있다.

```text
Personality
≈ characteristic deformation tendencies

DiachronicSelfness
≈ retained deformation history
 + transformation grammar
 + cross-time handoff
```

### 15.3 transformation grammar도 변한다

경험은 현재 morph뿐 아니라 매질의 물성도 바꾼다. 따라서 고정 불변 규칙 하나를 자아로 둘 수도 없다.

남는 후보는:

```text
perfect sameness
가 아니라
traceable handoff across lawful changes
```

이다.

이것은 Chapter 09의 Cross-Clock Selfness와 직접 비교할 질문이다.

```text

```

반례: 넓은 탐색이 유용한 탐색을 보장하지 않는다.

```text

→ easier future morph
```

봉인:

```text
easier thought
≠ chosen belief
```

소통 성공은 private form 동일성보다 translation·coordination의 충분성으로 평가한다.

Archive handoff가 전부 끊기면 현재 QualiaMorph와 Ghost candidate는 생길 수 있어도 이전 자아와의 통시적 연결은 약해진다.

```text

≠ distinct identity diagnosis
```

이다.

## 20. 이번 합성의 핵심 문장

```text

= prelinguistic deformable survival medium

QualiaMorph
= its currently felt, actionable form

Language
= a later anchor that stabilizes and reopens forms

Thought
= an endogenous experience that can reshape the medium
```

가장 압축하면:

> **생명은 외부 흐름을 그대로 견디는 것이 아니라, 퀄리아라는 변형 가능한 내부 매질에서 그 흐름을 먼저 형태로 살아본다. 생각은 그 매질의 자기 경험이고, 언어는 반복 가능한 형상을 붙잡는 후발 고정 장치다. 자아는 같은 형상을 유지하는 물체보다, 과거 변형을 이어받으며 새 세계에 맞게 다시 형태를 만드는 지속 문법에 가깝다.**
