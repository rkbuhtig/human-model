# Discarded Research

이 디렉터리는 현재 연구 방향에서 더 이상 사용하지 않는 이론, 종합, 개념 분해, 프롬프트 설계 아이디어를 보존한다.

폐기는 삭제와 다르다.

- 현재 모델이나 연구의 근거로 사용하지 않는다.
- 실패 이유를 가능한 한 남긴다.
- 무엇이 틀렸는지 아직 확실하지 않다면 그 불확실성도 기록한다.
- 전체가 틀렸더라도 일부 문제의식이나 표현은 이후 다시 회수할 수 있다.
- 다시 사용할 때에는 현재 연구인 것처럼 조용히 복원하지 않고, 어디서 왜 회수했는지 밝힌다.

권장 기록 항목은 다음과 같다.

```text
무엇을 설명하려 했는가
왜 유망하다고 생각했는가
어떤 모델 또는 평가와 연결되었는가
어디에서 실패했는가
현재 무엇이 이를 대체하는가
남겨둘 가치가 있는 부분은 무엇인가
다시 검토할 조건이 있는가
```

이 디렉터리는 실패를 숨기는 쓰레기통이 아니라, 같은 실패를 반복하지 않기 위한 연구 기억이다.
