# 가능성과 행동에서 다음에 구분해야 할 것

이 문서는 구조화된 가능성, 후보 생성, 선택, 수행 제한 조건이라는 설명이 실제로 필요한지 판별하기 위한 경쟁 모델과 질문을 모은다.

---

## 1. 결정적 정책과 확률적 정책

### 결정적 정책

현재 상태가 충분히 자세하면 다음 행동은 하나로 정해진다고 본다. 겉보기 변동은 관찰하지 못한 미세 상태와 환경 차이 때문이다.

### 확률적 정책

같은 상태와 맥락에서도 여러 행동이 실현될 수 있으며, 상태는 행동분포를 제한한다고 본다.

### 구분 질문

- 가능한 미세 조건을 최대한 맞춘 반복에서도 변동이 남는가?
- 변동을 설명하려고 상태를 계속 추가하면 모델이 과도하게 복잡해지는가?
- 확률적 모델이 held-out 상황에서 더 잘 일반화하는가?
- 확률을 도입했지만 실제로는 무작위 장식만 늘어난 것은 아닌가?

현재는 인간의 형이상학적 비결정성을 주장하지 않는다. 모델링 단위에서 분포가 더 적절한지 묻는다.

---

## 2. 직접 정책과 후보–gate 설명

### 직접 정책

```text
state + context
→ action distribution
```

중간 후보를 두지 않는다.

### 후보–gate 설명

```text
state + context
→ candidate set
→ shaping / gate
→ action distribution
```

### 후보–gate가 필요한 조건

- 생성되는 가능성 폭과 최종 선택을 독립적으로 바꾸는 개입이 가능하다.
- 같은 최종 행동이라도 고려했던 대안이 이후 후회와 학습을 바꾼다.
- capacity 저하가 가능성 생성과 선택·수행에 다른 영향을 준다.
- 금지된 생각이 떠오르지만 행동 준비와 수행으로 가지 않는 현상을 별도로 설명해야 한다.

### 후보–gate를 폐기할 조건

직접 정책이 같은 개입 반응과 이후 피드백을 더 단순하게 예측한다.

### 모델 언어 경계

후보–gate는 우선 LLM과 프롬프트 엔진에서 회수한 비교 언어다. 인간이 실제로 candidate set과 gate를 순서대로 갖는다는 주장은 아니다.

---

## 3. 후보 언어의 소급 위험

LLM이 여러 토큰 가능성을 계산하고 하나를 출력한다는 사실 때문에 인간도 명시적 후보 집합을 생성한다고 가정할 위험이 있다.

### 점검 질문

- 후보라는 말을 쓰지 않고도 같은 현상을 정책, affordance, 행동 준비, 습관으로 설명할 수 있는가?
- 후보가 관찰 가능한 차이를 만들지 않고 설명용 은유로만 남는가?
- 후보의 개수와 경계가 모델 설계자 임의로 정해지는가?
- 인간 현상보다 LLM decoding 구조를 더 잘 설명하고 있지는 않은가?
- 미선택 후보가 이후 수행에 흔적을 남기지 않아도 후보를 유지할 이유가 있는가?

후보 개념은 여러 가능성이 동시에 접근 가능하고 이후 과정에 선택적으로 다른 흔적을 남길 때만 유지할 가치가 있다.

---

## 4. 다양성과 일관성

같은 모델을 여러 번 실행했을 때 다음 두 실패를 구분한다.

### Mode collapse

- 늘 이상적으로 차분함
- 갈등마다 확인과 대화를 선택함
- 피로와 압박에도 같은 깊이로 심의함
- 관계와 역사에 상관없이 안전한 반응으로 수렴함

### 무구조한 randomness

- 상태를 바꿔도 분포가 거의 같음
- 핵심 욕구와 정반대인 행동이 이유 없이 빈번함
- 작은 환경 잡음이 정체성 수준의 변화를 만듦
- 장기 경향이나 복귀 구조가 없음

### 필요한 판별

좋은 분포는 단순히 중간 엔트로피를 갖는 것이 아니다.

```text
관련 상태 변화에는 민감함
무관한 표현 변화에는 비교적 안정적임
작은 교란에는 제한된 변동
구조적 사건에는 분포 이동
장기적으로 반복되는 범위와 경향
```

을 보여야 한다.

## 5. 생성과 형상화의 독립성

### 사례 A

불안이 높아져 새로운 위험 해석이 더 많이 떠오른다.

### 사례 B

위험 해석의 종류는 같지만 행동 준비와 수행 가능성만 높아진다.

### 사례 C

공개 hazard 때문에 같은 생각을 하면서도 표면 표현만 제한된다.

### 질문

- 생성되는 가능성의 종류와 상대 접근성을 구분할 수 있는가?
- 내부 생각과 표면 행동분포를 별도로 관찰할 수 있는가?
- 같은 생각이 있어도 행동 준비나 수행만 달라지는 현상이 있는가?
- 하나의 정책 변화가 이 모든 차이를 충분히 설명하는가?

## 6. 충동·습관·심의

### 경쟁 설명 A — 별도 경로

각각 다른 입력, 학습, 수행 조건, 사후 평가를 가진다.

### 경쟁 설명 B — 하나의 연속체

같은 선택 과정이 capacity와 clock에 따라 빠르고 얕거나 느리고 깊게 작동한다.

### 경쟁 설명 C — 관찰자의 사후 분류

행동 뒤 이유와 속도를 보고 충동·습관·심의라고 이름 붙일 뿐이다.

### 판별 질문

- 결과가 예상과 다를 때 업데이트 방식이 다른가?
- 심의 시간을 늘리면 충동이 연속적으로 변하는가, 다른 경로로 전환되는가?
- 습관은 현재 목표가 바뀌어도 유지되는가?
- 행동 후 저자성 관계와 책임 인수가 수행 경로에 따라 달라지는가?
- 같은 clock에서도 capacity만 바꾸면 다른 결과가 나오는가?

---

## 7. 수행 제한 조건의 작용점

`resource pressure` 하나로 묶지 않고 네 종류를 비교한다.

### Capacity

```text
후보 종류와 폭
비교 깊이
익숙한 반응 억제
행동 실행 성공률
표현 통제
결과를 견디고 처리하는 능력
```

### Clock

```text
탐색에 허용된 시간
보류 가능 시간
사건 사이 회복 간격
학습과 재구성에 필요한 실제 시간
```

### Stake / Hazard

```text
관계 손실
거절과 수치
공개와 평판
갈등과 처벌
실제 손상 가능성
```

### Option Constraint

```text
되돌리기 비용
현재 선택이 미래 경로를 닫는 정도
commitment
수리와 재접근 가능성의 폐쇄
```

### 판별 질문

- 피로와 짧은 시간 제한이 같은 방식으로 후보 폭을 줄이는가?
- 관계 hazard를 높이면 내부 생각과 표면 표현 가운데 무엇이 먼저 바뀌는가?
- 현재 위험은 같지만 미래 option loss만 다르면 선택이 달라지는가?
- 휴식은 capacity를 회복시키지만 이미 닫힌 관계 경로는 복구하지 못하는가?
- 네 종류를 하나의 scalar로 환산하면 선택적 개입 차이가 사라지는가?

이 구분이 여러 thread에서 안정적인 공통 법칙을 보이면 별도 문제 공간으로 분리할 수 있다. 그렇지 않으면 각 thread의 context 변수로 돌려보낸다.

---

## 8. 선택되지 않은 가능성

### 비교 사례

두 사람이 같은 행동을 선택했지만:

```text
A는 다른 대안을 거의 고려하지 않았다.
B는 관계를 끊고 싶었지만 유지하기로 선택했다.
```

같은 행동 뒤에도 후회, 피로, 자기평가, 다음 선택이 다를 수 있다.

### 질문

- 미선택 대안이 이후 행동을 예측하는가?
- 대안 고려 여부를 실제로 추론할 수 있는가?
- 미선택 후보를 저장하지 않고 현재 갈등 상태나 접근 편향으로 압축할 수 있는가?
- 모든 대안을 보존하면 모델이 사후 설명에 과적합하지 않는가?

---

## 9. 내부 가능성과 표면 행동

같은 내부 생각과 행동 준비가 역할, 공개성, 관계 hazard에 따라 다른 표면 표현을 만들 수 있다.

### 질문

- 표면 표현만으로 내부 candidate set을 역추론할 수 있는가?
- 공개성만 바꾸었을 때 내부 생각은 유지되고 표현만 달라지는가?
- 표현 정책을 일반 행동 선택에 포함하면 별도 surface 구분이 불필요한가?
- 숨김과 누출이 가능성 연구가 아니라 `self-other-surface`로 이동해야 하는가?

## 10. 현재 설명의 실패 조건

### 실패 A

직접 state-to-action 정책이 후보–gate 모델과 동일한 개입 결과를 더 단순하게 만든다.

### 실패 B

반복 실행의 다양성이 seed나 표현 잡음일 뿐 상태와 역사에 따라 구조적으로 변하지 않는다.

### 실패 C

충동·습관·심의 구분이 사후 언어 라벨 외에 다른 학습과 수행을 예측하지 않는다.

### 실패 D

Capacity, clock, stake, option constraint를 나눠도 선택적으로 다른 결과가 나타나지 않는다.

### 실패 E

미선택 가능성이 이후 수행에 아무 차이도 만들지 않는다.

### 실패 F

후보 개념이 인간 현상보다 LLM decoding 구조를 설명하는 데만 유용하다.

### 실패 G

생각 가능성, 행동 준비, 선택, 수행을 분리해도 관측과 개입에서 차이가 없다.

---

## 11. 이 thread를 분리할 신호

- 가능성 생성과 최종 선택을 독립적으로 조작할 수 있다.
- 수행 제한 조건들이 여러 과정에서 공통된 별도 법칙을 보인다.
- 습관이 일반 선택과 다른 학습·실패 구조를 가진다.
- 표면 표현 선택이 내부 행동 준비와 독립적으로 움직인다.
- 미선택 가능성이 장기 피드백의 독립 원인이 된다.

그 전에는 현재의 넓은 문제 묶음을 유지한다.
