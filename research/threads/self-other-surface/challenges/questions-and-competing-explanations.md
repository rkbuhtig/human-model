# 자기, 타자, 표면에서 다음에 구분해야 할 것

이 문서는 내부 상태–표현 비동일성, 타자 모델, 관계 상태라는 설명이 실제로 필요한지 판별하기 위한 경쟁 모델과 질문을 모은다.

---

## 1. 단순 부분관측으로 충분한가

### 설명

내부 상태는 직접 보이지 않고 표면 행동만 관측되므로, 같은 출력에서 여러 내부 상태가 가능하다는 것은 일반적인 부분관측 문제일 뿐이라고 본다.

### 현재 설명이 추가하려는 것

- 역할과 공개성이 표현 정책을 선택적으로 바꿈
- 숨김 의도와 비의도적 누출이 함께 존재함
- 타인의 예상 반응이 표현을 형상화함
- 같은 내부 상태도 관계별로 다른 표면을 만듦

### 판별 질문

- 일반적인 observation noise만으로 역할·공개성 개입을 설명할 수 있는가?
- 숨김과 누출을 별도 메커니즘 없이 하나의 stochastic emission으로 충분히 표현할 수 있는가?
- 타자 모델을 제거해도 같은 상황적 표현 변화가 유지되는가?

단순 emission model이 더 적은 가정으로 같은 수행을 내면 projection 언어를 축소해야 한다.

---

## 2. 투영과 전략적 행동

### 투영 설명

표현은 내부 상태가 제약을 거쳐 부분적으로 드러난 결과다.

### 전략적 행동 설명

표현은 원하는 결과를 얻기 위해 선택된 행동이다.

### 구분 사례

```text
상대를 보호하기 위해 감정을 숨김
평판을 지키기 위해 침착한 척함
숨기려 했지만 말투에서 감정이 샘
감정은 약하지만 관계 전략으로 강하게 표현함
```

### 질문

- 표현의 방향이 내부 감정과 반대여도 기능적으로 성공하는가?
- 통제 자원을 낮추면 전략은 유지되고 누출만 증가하는가?
- 의도한 표현과 실제 표면을 따로 추론할 필요가 있는가?
- 같은 행동 정책으로 투영과 전략을 모두 설명할 수 있는가?

---

## 3. 실제 타인과 내부 타자 모델

### 최소 사례

A는 B가 자신을 싫어한다고 믿지만, B의 실제 행동은 지속적으로 친밀성과 신뢰를 보여준다.

### 가능한 경로

```text
A가 모델을 수정함
B의 행동을 예외로 처리함
B의 행동을 위선 또는 연민으로 재해석함
A의 자기상이 바뀌어 B 모델도 함께 바뀜
```

### 질문

- 어떤 종류의 반증이 타자 모델을 수정하는가?
- 단일 사건과 반복 증거의 효과가 다른가?
- 신뢰하는 제3자의 정보는 직접 행동과 다르게 작동하는가?
- 기존 Narrative가 반증을 얼마나 흡수하는가?

---

## 4. 재귀적 추정의 필요성

### 1차 모델

```text
A가 생각하는 B
```

### 2차 모델

```text
A가 생각하는 “B가 보는 A”
```

### 질문

- 2차 모델이 없으면 설명할 수 없는 표현 변화가 있는가?
- 공개적 창피함, 체면, 자기검열은 1차 상대 모델로 충분한가?
- 재귀 깊이를 늘릴수록 실제 예측이 나아지는가 아니면 사후 설명만 늘어나는가?
- 사람은 명시적 재귀 모델이 아니라 단순한 사회적 위험 신호만 사용할 수 있는가?

재귀 깊이가 별도 수행 차이를 만들지 않으면 압축된 사회 기대 변수로 대체해야 한다.

## 5. 관계의 residence

### 후보 A — 개인 내부 상태

A와 B가 각각 관계 신뢰와 기대를 보유한다.

### 후보 B — 공유된 상호작용 기록

관계는 실제로 오간 행동과 합의의 역사다.

### 후보 C — 결합 상태

A와 B가 연결될 때만 성립하는 공동 동역학이다.

### 후보 D — 관찰자 요약

관계라는 것은 반복 상호작용을 나중에 붙인 요약이다.

### 판별 질문

- A와 B의 관계 모델이 충돌할 때 관계의 현재 상태는 무엇인가?
- 한쪽만 화해했다고 느끼면 관계는 회복된 것인가?
- 공유 기록만 같아도 서로 다른 관계 모델이 다른 행동을 만드는가?
- 두 사람을 분리해도 관계 상태가 개인 내부에 유지되는가?

## 6. 역할 변화와 내부 변화

### 비교 사례

같은 사람이 사적 상황에서는 솔직하지만 공개 상황에서는 냉정하게 행동한다.

### 설명 A

내부 상태는 같고 표현 정책만 다르다.

### 설명 B

역할이 활성화되면 실제 욕구와 자기상도 함께 바뀐다.

### 설명 C

반복 역할 수행이 장기적으로 내부 상태를 형성한다.

### 질문

- 역할이 종료되면 즉시 이전 표현으로 돌아오는가?
- 장기간 역할 수행 뒤 사적 상황에서도 변화가 남는가?
- 역할에 맞는 행동 뒤 자기 피드백이 자기상을 수정하는가?
- 역할 변화와 실제 정체성 변화를 어떻게 구분할 것인가?

## 7. 눈치·체면·누출의 문화적 특수성

### 위험

한국어 생활 세계에서 나온 개념을 보편 인간 구조로 과잉 일반화할 수 있다.

### 기능적 재기술 후보

```text
눈치
→ 타인의 상태와 예상 반응에 대한 빠른 추정 및 행동 조정

체면
→ 공개 평가와 관계 지위에 따른 표현 제약

누출
→ 표현 통제와 실제 표면 사이의 불완전성
```

### 질문

- 다른 문화권에서도 같은 기능이 다른 표현으로 나타나는가?
- 개인차가 문화차보다 큰가?
- 명명만 문화 특수적이고 기능은 일반적인가?
- 이 기능들이 독립적으로 필요한가, 일반 사회적 추론으로 충분한가?

## 8. 같은 표현, 다른 내부 상태

다음은 모두 같은 “괜찮아”라는 표현을 만들 수 있다.

```text
실제로 괜찮음
화를 숨김
관계를 끊기로 해서 더 말할 필요가 없음
상대를 보호하려 함
갈등을 피할 자원이 없음
감정 자체를 아직 파악하지 못함
```

### 질문

- 후속 행동과 시간적 잔여가 내부 상태를 구분하는가?
- 단일 표면에서 내부를 확정하려는 평가가 왜 위험한가?
- 여러 미래 probe를 통해서만 구분 가능한가?
- 표면을 지나치게 숨기면 모델이 아무 내부 상태나 주장할 수 있지 않은가?

## 9. 현재 설명의 실패 조건

### 실패 A

상태조건부 stochastic emission 하나가 역할·숨김·누출을 충분히 설명한다.

### 실패 B

타자 모델을 명시하지 않은 직접 정책이 상대 반응 개입에 동일하게 적응한다.

### 실패 C

재귀적 타자 추정이 1차 기대 변수보다 추가 예측을 만들지 않는다.

### 실패 D

관계 상태가 개인 내부 상태와 공유 기록으로 완전히 환원된다.

### 실패 E

눈치·체면·누출 구분이 문화적 표현을 다시 이름 붙인 것 외에 수행 차이를 만들지 않는다.

### 실패 F

표현과 행동 선택을 분리해도 실제 모델 구현과 평가에서 구분할 수 없다.

## 10. 이 thread를 분리할 신호

- 타자 모델 업데이트가 표현 없이도 독립된 수행 차이를 만든다.
- 역할·공개성에 따른 표현 정책이 타자 모델과 독립적으로 움직인다.
- 관계가 개인 내부와 공유 기록으로 환원되지 않는 결합 상태를 요구한다.
- 숨김과 누출이 일반 stochastic emission 이상의 구조를 보인다.
- 문화 특수성과 일반 기능의 비교가 독립 연구가 된다.

그 전에는 현재의 넓은 묶음을 유지한다.
