# 자기 피드백에서 다음에 구분해야 할 것

이 문서는 자기 피드백이라는 이름이 너무 많은 사후 변화를 흡수하지 않도록 경쟁 설명과 폐기 조건을 모은다.

---

## 1. 일반 강화학습으로 충분한가

### 강화 설명

좋은 결과는 행동 가능성을 높이고 나쁜 결과는 낮춘다.

### 현재 설명이 추가하려는 것

- 어떤 결과를 자기 행동과 연결하는가
- 결과와 자기 목표의 충돌을 어떻게 해석하는가
- 후회·수치·합리화가 왜 다른 경로를 만들 수 있는가
- 책임 인수와 구체적 수리가 어떻게 갈리는가
- 타인의 용서와 자기 변화가 왜 분리되는가
- 현재 감정과 장기 행동 변화가 왜 함께 움직이지 않을 수 있는가

### 판별 질문

- 단순 보상 신호만으로 책임 인수와 관계 수리를 예측할 수 있는가?
- 같은 부정 결과에서도 후회·수치·비난 전가가 다른 행동을 만드는가?
- 외부 보상이 같아도 자기 기준과의 충돌에 따라 학습이 달라지는가?
- 강화 모델에 상태를 계속 추가하는 것과 별도 자기평가 설명 사이에 실질적 차이가 있는가?
- 타인의 경계와 명시적 합의를 보상 값 안에 넣으면 규범 문제를 심리 상태로 잘못 흡수하지 않는가?

일반 강화학습 모델이 더 단순하고 같은 선택적 예측을 만들면 현재 thread는 축소한다.

`reinforcement`, `prediction error`, `policy update`는 비교 모델 언어다. 인간 경험이 실제로 하나의 보상 신호로 구현된다는 주장은 아니다.

---

## 2. 후회 표현과 실제 변화

### 최소 비교

```text
A. 후회를 길게 설명하지만 같은 상황에서 같은 행동을 반복함
B. 후회를 거의 말하지 않지만 다음 행동에서 확인과 억제가 생김
```

### 질문

- 어느 쪽을 더 강한 자기 피드백으로 볼 것인가?
- 언어 표현과 행동 변화를 별도 readout으로 봐야 하는가?
- 행동이 바뀌지 않은 후회는 거짓인가, capacity·clock·습관 때문에 실현되지 못한 것인가?
- 같은 상황 재현 없이 후회의 작동을 판정할 수 있는가?
- 수리 행동이 없더라도 현재 지지 철회나 책임 인수는 실제일 수 있는가?

---

## 3. 후회와 수치

### 경쟁 설명 A

두 현상은 같은 부정 정서의 강도 차이다.

### 경쟁 설명 B

후회는 행동과 선택을 평가하고 수치는 자기 전체와 공개된 자기를 평가한다.

### 경쟁 설명 C

차이는 감정 자체보다 예상되는 타인의 평가, 공개성, 관계 hazard에서 생긴다.

### 판별 질문

- 같은 정서 강도에서 수리 행동과 은폐 행동이 갈리는가?
- 타인에게 드러나지 않은 상황에서도 수치 경로가 나타나는가?
- 자기 전체 비난을 줄이면 책임 인수는 높아지는가 낮아지는가?
- 후회와 수치의 언어 라벨을 바꿔도 실제 행동분포가 유지되는가?
- 공개 hazard만 높였을 때 행동 평가보다 자기 은폐가 증가하는가?

---

## 4. 합리화와 적응적 재해석

두 과정 모두 사건의 현재 의미를 바꿀 수 있다.

### 방어적 합리화 후보

- 외부 반증을 무시함
- 자신의 인과적 참여와 책임만 선택적으로 축소함
- 타인의 경험과 경계를 지움
- 기존 행동을 유지하기 위한 해석 변경

### 적응적 재해석 후보

- 새 정보를 반영함
- 인과적 참여, 통제, 실제 책임 범위를 구분함
- 과도한 자기비난과 실제 의무를 분리함
- 발생 사실과 타인의 손상을 보존함
- 더 나은 미래 선택과 수리 가능성으로 이어짐

### 질문

- 미래 행동이 나아졌다는 이유만으로 해석이 사실적이라고 볼 수 있는가?
- 외부 사실을 보존하면서 자기 책임을 다르게 배분할 수 있는가?
- 자기 안정화와 사실 왜곡을 어떤 기준으로 구분할 것인가?
- 타인의 경계를 보존하지만 자기비난을 줄이는 재해석이 가능한가?
- 해석 변경 없이 정책만 바뀌어도 같은 적응이 가능한가?

---

## 5. 사과, 책임 인수, 수리

### 비교 사례

```text
A. 즉시 사과하지만 구체적 행동은 바뀌지 않음
B. 사과 표현은 서툴지만 손상 복구와 재발 방지를 실행함
C. 사과와 행동 변화는 있으나 상대의 경계를 존중하지 않음
D. 책임은 인정하지만 상대가 원하지 않는 수리를 강요함
```

### 질문

- 사과는 내부 상태의 증거인가 사회적 행동인가?
- 수리 행동 없이도 책임 인수가 가능한가?
- 책임 인수 없이 전략적 수리 행동만 수행할 수 있는가?
- 상대가 수리를 원하지 않을 때 책임 있는 행동은 무엇인가?
- 관계 회복 욕구가 상대 통제로 바뀌는 경계는 어디인가?
- 행위자의 변화와 실제 관계적·규범적 정산은 어떻게 분리되는가?

---

## 6. 자기 변화와 타인의 반응

### 경쟁 설명 A — 자기평가 중심

자신의 기준과 결과의 불일치가 핵심이다.

### 경쟁 설명 B — 사회적 피드백 중심

타인의 실제 반응과 관계 결과가 핵심이다.

### 경쟁 설명 C — 복수 입력 설명

자기 기준, 실제 타인의 반응, 실제 interaction history, 상호 인정, 명시적 합의가 서로 다른 입력을 제공한다.

### 판별 질문

- 타인의 반응이 없어도 자기 변화가 생기는가?
- 타인이 용서하면 문제 행동이 오히려 강화될 수 있는가?
- 사회적 비난이 커도 자신이 행동을 지지하면 정책은 유지되는가?
- 자기 변화가 실제라도 타인의 경계를 침범할 수 있는가?
- 같은 타인 반응을 각자가 다르게 기록하면 학습도 달라지는가?
- 명시적 합의 위반이 개인 감정과 별도로 행동을 바꾸는가?

---

## 7. 현재 상태, 중간 잔여, 장기 적응

### 최소 비교

```text
A. 사건 직후 강한 후회, 며칠 뒤 원래 행동으로 복귀
B. 감정 반응은 작지만 이후 유사 상황에서 행동이 지속적으로 달라짐
C. 현재 행동은 바뀌었지만 높은 hazard가 사라지면 원래 경로로 돌아감
```

### 질문

- 강한 현재 감정과 장기 학습을 분리해야 하는가?
- 반복된 작은 피드백이 한 번의 큰 사건보다 전이 민감도를 더 많이 바꿀 수 있는가?
- 장기 변화가 현재 상태에 압축되는가, 별도 정책 또는 transition 언어가 필요한가?
- 중간 잔여와 장기 적응을 선택적으로 변화시키는 개입이 가능한가?
- 조건이 회복된 뒤에도 변화가 남는가?

## 8. Capacity, clock, hazard, option constraint

행동이 바뀌지 않았다는 사실만으로 자기평가와 변화 의도가 없었다고 판정하지 않는다.

### 비교할 가능성

```text
capacity 부족
→ 새 선택을 유지하거나 수행하지 못함

짧은 clock
→ 익숙한 행동이 먼저 실행됨

높은 관계 hazard
→ 수리보다 은폐와 회피를 선택함

option constraint
→ 현재 수리가 미래 관계 경로를 닫을까 보류함
```

### 질문

- capacity를 회복시키면 같은 자기평가가 실제 행동 변화로 이어지는가?
- 시간을 늘리면 후회와 책임 인수는 같고 수행만 달라지는가?
- 공개 hazard를 낮추면 사과 표현보다 구체적 수리가 증가하는가?
- 미래 경로 폐쇄 위험을 낮추면 보류했던 행동을 수행하는가?

이 구분은 네 개의 인간 내부 모듈을 가정하지 않는다. 후회 표현과 행동 변화 사이의 실패를 한 단어로 설명하지 않기 위한 비교 조건이다.

---

## 9. 회복의 방향

회복을 단순히 이전 상태로 돌아가는 것으로 두면 다음 문제가 생긴다.

```text
회피 습관으로 돌아감
자기기만을 통해 정서 안정
관계 손상은 남았지만 내부 불편만 감소
```

### 질문

- 회복은 누구의 어떤 기능을 기준으로 하는가?
- 자기 안정과 관계 수리가 충돌할 때 무엇을 회복이라고 부르는가?
- 이전 상태가 방어적이면 복귀는 회복인가?
- 새로운 경로를 배우는 적응과 기존 basin 복귀를 어떻게 구분할 것인가?
- 타인의 관계 회복 거부가 개인 내부 적응을 무효화하는가?

## 10. 서로 다른 문장 종류

### 서술 질문

결과와 자기평가가 이후 상태와 행동을 어떻게 바꾸는가?

### 사실 질문

무슨 행동과 손상이 실제로 발생했는가?

### 규범 질문

어떤 책임과 수리가 요구되며 타인의 권리와 경계는 무엇인가?

### 관측 질문

표현, 반복 행동, 수리, 장기 변화 가운데 무엇으로 피드백의 작동을 판정할 수 있는가?

규범적으로 바람직한 변화와 실제 인간 동역학을 같은 주장으로 사용하지 않는다.

---

## 11. 현재 설명의 실패 조건

### 실패 A

보상과 단순 정책 업데이트만으로 모든 핵심 사례가 설명된다.

### 실패 B

후회·수치·합리화 구분이 이후 행동이나 정보 처리에 다른 예측을 만들지 않는다.

### 실패 C

언어적 자기평가와 실제 행동 변화가 항상 함께 움직여 별도 구분이 불필요하다.

### 실패 D

사회적 피드백 없이 자기 피드백이라는 과정이 성립하지 않는다.

### 실패 E

현재 상태, 중간 잔여, 장기 adaptation을 나눠도 선택적 예측 차이가 없다.

### 실패 F

책임과 수리가 `events-and-change`의 정산 문제로 완전히 흡수된다.

### 실패 G

Capacity, clock, hazard, option constraint를 나눠도 행동 변화 실패를 더 잘 설명하지 못한다.

---

## 12. 이 thread를 분리할 신호

- 정서 반응과 책임 인수가 독립적인 예측을 만든다.
- 자기평가와 사회적 피드백이 서로 다른 학습 경로를 갖는다.
- 현재 상태, 중간 잔여, 장기 adaptation이 분명히 갈린다.
- 수리 행동이 일반 선택이나 표현과 다른 구조를 요구한다.
- 합리화와 재저자화의 경계가 독립 연구 문제로 커진다.
- 관계적·규범적 정산이 개인 자기 피드백으로 환원되지 않는다.

이 신호가 생기기 전에는 `self-feedback`을 넓은 작업 묶음으로 유지한다.
