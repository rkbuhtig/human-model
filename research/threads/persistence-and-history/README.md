# Persistence and History

이 연구 흐름은 **무엇이 시간 속에서 남고, 과거가 어떤 형태로 현재와 미래에 작동하는가**를 조사한다.

현재 기억, 잔여, Episode, Narrative, 습관, 전이 경사, 재해석을 독립 객체로 확정하지 않는다.
같은 현상을 다른 이름으로 부르고 있는지, 하나의 이름 아래 여러 기능이 섞였는지 먼저 본다.

---

## 1. 현재 중심 질문

```text
한 장면이 끝난 뒤 무엇이 남는가?
남은 것은 내용, 상태, 접근 편향, 전이법 가운데 무엇인가?
실제 시간이 흐르는 것과 사건이 반복되는 것은 어떻게 다른가?
작은 사건은 왜 사라지고 일부 사건은 오래 남는가?
반복·누적·점화·회복·고착은 어떤 시간 구조를 갖는가?
Episode와 Narrative라는 구분이 실제로 필요한가?
과거의 의미가 바뀌어도 발생 사실이 유지될 수 있는가?
```

---

## 2. 현재 함께 다루는 문제

### 지속

현재 상태가 다음 장면까지 얼마나, 어떤 형태로 남는가.

### 실제 시간과 상호작용 시간

아무 사건이 없어도 피로와 감정이 변하는가.
같은 사건 수라도 짧은 시간과 긴 시간에 걸쳐 일어난 경우가 다른가.

### 잔여

사건의 내용 전체가 아니라 미해결 긴장, 기대, 불신, 애착, 책임 같은 효과가 남을 수 있는가.

### 기억

과거를 다시 말할 수 있는 것과, 과거가 현재의 주의·해석·행동을 기울이는 것은 어떻게 다른가.

### Episode

다음 후보들을 비교한다.

```text
실제로 걸은 사건–해석–행동–결과 경로
그 경로가 남긴 중간 시간척도의 상태
나중에 다시 접근하는 기록이나 포인터
여러 사건이 하나의 의미 단위로 묶인 것
```

### Narrative

다음 후보들을 비교한다.

```text
현재 활성화된 자기서사 내용
여러 Episode를 거쳐 생긴 느린 상태
전이확률이나 접근 경사의 변화
반복 경로를 요약한 graph
관계 안에서 공유되거나 충돌하는 해석
```

### 경로 의존성과 복귀

같은 현재 상태처럼 보여도 어떤 경로로 왔는지에 따라 이후 반응이 달라지는가.
작은 교란 뒤 어디로 돌아가며, 큰 사건 뒤 무엇이 바뀌는가.

---

## 3. 현재 포함하지 않는 문제

- 발생과 책임의 확정: `events-and-change`
- 행동 가능성의 생성과 선택: `possibility-and-action`
- 결과가 변화를 만드는 구체 과정: `self-feedback`
- 타자 모델과 표현: `self-other-surface`

그러나 시간적 지속은 모든 흐름을 건드리므로 중첩을 지도에 기록한다.

---

## 4. 현재 경쟁하는 설명

### 내용 저장 모델

과거의 내용을 보존하고 필요할 때 다시 읽는 것이 기억의 핵심이라고 본다.

### 지속 상태 모델

과거 사건이 현재 상태 일부로 남아 다음 반응을 직접 바꾼다고 본다.

### 접근 편향 모델

과거가 무엇을 먼저 떠올리고 어떻게 해석할지를 기울인다고 본다.

### 전이법 변화 모델

반복된 과거가 현재값이 아니라 상태 변화의 법칙 자체를 바꾼다고 본다.

### 경로 요약 모델

Episode와 Narrative는 원인 객체가 아니라 실제 궤적을 나중에 요약한 readout이라고 본다.

이 설명들이 동시에 일부 맞을 수도 있지만, 같은 이름으로 뭉치지 않고 기능별 차이를 본다.

---

## 5. 중요한 구분

```text
기억 내용 ≠ 현재 잔여
현재 잔여 ≠ 전이법 변화
실제 경로 ≠ 경로의 기록
Episode record ≠ Episode residue
Narrative state ≠ Narrative description
복귀 ≠ 건강한 회복
지속 ≠ 권한
오래 남음 ≠ 사실임
```

---

## 6. 이 thread가 독립적이지 않을 가능성

- 지속은 모든 상태의 일반 속성이므로 별도 문제 공간이 불필요할 수 있다.
- Episode와 Narrative는 `self-feedback`의 장기 효과에 불과할 수 있다.
- 기억의 미래 편향은 `possibility-and-action`의 후보 형상화로 환원될 수 있다.
- 관계적 Narrative는 `self-other-surface`에서 다루는 편이 나을 수 있다.
- 경로 의존성처럼 보이는 현상이 누락된 현재 상태 변수 때문일 수 있다.

---

## 7. 분리 조건

### 기억과 지속 분리

재접근 가능한 내용과 비언어적 잔여가 독립적으로 움직이는 경우.

### Episode 분리

경로·기록·잔여가 서로 다른 메커니즘과 예측을 가질 경우.

### Narrative 분리

느린 상태, 전이법 변화, graph summary, 자기설명이 실제로 독립적인 경우.

### 시간축 분리

실제 시간에 따른 변화와 사건 반복에 따른 변화가 다른 법칙을 가질 경우.

---

## 8. 병합 조건

- 장기 변화가 전부 `self-feedback`의 학습 과정으로 설명됨
- 기억 편향이 전부 `possibility-and-action`의 제약 형상화로 설명됨
- Narrative가 개인 내부보다 관계 모델의 산물로 드러나 `self-other-surface`와 합치는 편이 나음

---

## 9. 개명 조건

핵심이 보존보다 변화율과 경로 의존성으로 이동하면 다음 이름을 검토한다.

```text
temporal-dynamics
history-dependent-change
residue-and-adaptation
```

---

## 10. 해체 조건

- 시간 관련 문제라는 이유만으로 서로 다른 현상을 과도하게 묶음
- Episode와 Narrative 용어가 질문보다 먼저 구조를 고정함
- 모든 문제가 다른 흐름의 시간적 속성으로 충분히 설명됨

---

## 11. 현재 우선 작업

1. Episode trajectory, residue, record를 구분하는 사례를 만든다.
2. Narrative의 네 후보 의미가 실제 수행에서 어떤 차이를 만드는지 적는다.
3. 같은 사건 수와 다른 실제 시간 간격을 비교한다.
4. 작은 교란 후 복귀, 반복 누적, 점화, 장기 drift를 하나의 회복 서사로 뭉치지 않는다.
5. 전체 과거를 제거하고 현재 잔여만 남겼을 때 미래 반응이 유지되는지 생각한다.
