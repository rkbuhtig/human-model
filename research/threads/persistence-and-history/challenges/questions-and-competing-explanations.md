# 지속과 역사에서 다음에 구분해야 할 것

이 문서는 시간·기억·Episode·Narrative에 대한 현재 설명이 무엇과 경쟁하며 어떤 관찰 앞에서 바뀌어야 하는지 정리한다.

---

## 1. 내용 저장 모델과 잔여 상태 모델

### 비교 사례

과거 갈등의 세부 내용은 거의 기억하지 못하지만, 같은 상대 앞에서 긴장과 회피가 남아 있다.

### 내용 저장 모델의 설명

- 기억 내용이 불완전하게 남아 있다.
- 현재 반응은 암묵적으로 검색된 과거 정보의 결과다.

### 잔여 상태 모델의 설명

- 사건 내용과 별개로 긴장·불신·행동 준비가 현재 상태에 남았다.

### 구분 질문

- 과거 내용을 수정해도 현재 반응이 유지되는가?
- 세부 회상 없이도 반응의 방향이 안정적으로 남는가?
- 잔여를 낮추면 기억 내용은 그대로여도 행동이 달라지는가?
- 기억을 반복 설명하는 것과 실제 접근·회피 변화가 분리되는가?

---

## 2. 잔여 상태와 접근 편향

### 최소 차이

```text
잔여 상태
= 현재 긴장·불신이 실제로 활성화되어 있음

접근 편향
= 평소에는 드러나지 않지만 유사 사건에서 특정 해석이 먼저 열림
```

### 구분 질문

- 관련 자극이 없을 때에도 차이가 유지되는가?
- 같은 현재 감정에서도 어떤 해석이 먼저 떠오르는지가 다른가?
- 주의 또는 검색만 조작했을 때 행동 준비까지 바뀌는가?

---

## 3. 현재 상태와 전이법 변화

### 비교 사례

두 사람 모두 현재 신뢰 수준이 비슷해 보이지만, 한 사람은 작은 위반 뒤 급격히 무너지고 다른 사람은 천천히 변한다.

### 설명 A

실제로 현재 상태가 같지 않으며 민감도 변수가 빠져 있다.

### 설명 B

현재값은 같지만 transition kernel이 다르다.

### 설명 C

장기 역사 전체를 봐야 하며 현재 압축 상태로는 충분하지 않다.

### 판별 질문

- 여러 종류의 작은 개입에서 변화율 차이가 일관되게 나타나는가?
- 민감도를 현재 상태에 넣으면 과거 전체 없이 미래를 예측할 수 있는가?
- 새로운 경험 뒤 민감도 자체가 학습되는가?

---

## 4. Episode trajectory, record, residue

동일한 사건 구간에 대해 다음을 따로 관찰한다.

```text
실제로 어떤 경로를 걸었는가
나중에 무엇을 회상할 수 있는가
현재 어떤 효과가 남아 있는가
어디에서 한 구간이 끝났다고 느끼는가
```

### 분리 신호

- 같은 trajectory인데 record가 다르다.
- 같은 record인데 residue가 다르다.
- residue가 사라졌지만 기록은 남는다.
- 당사자와 관찰자가 Episode 경계를 다르게 나눈다.

이 신호가 반복되면 Episode를 하나의 객체로 쓰는 것을 중단해야 한다.

---

## 5. Narrative 경쟁 모델

### 모델 A — 언어 내용

Narrative는 자신과 관계에 대해 현재 말할 수 있는 이야기다.

### 모델 B — 느린 상태

Narrative는 장기 기대·신뢰·자기상처럼 지속하는 내부 상태다.

### 모델 C — 전이법

Narrative는 어떤 사건에 민감하고 어디로 복귀하는지를 바꾸는 변화 규칙이다.

### 모델 D — 경로 요약

Narrative는 실제 trajectory 집합을 나중에 압축해 표현한 readout이다.

### 모델 E — 관계적 복수 모델

A와 B에게 서로 다른 Narrative가 있고 공유된 하나의 Narrative는 없다.

### 판별 개입

- 자기설명만 바꾸고 실제 경험은 그대로 둔다.
- 장기 기대만 바꾸고 자기설명은 유지한다.
- 같은 현재 상태에서 전이 민감도만 다르게 둔다.
- 같은 경로를 서로 다른 이야기로 요약한다.
- A의 관계 이야기와 B의 관계 이야기를 충돌시킨다.

각 모델이 이후 주의·행동·복귀에 다른 예측을 내는지 본다.

---

## 6. attractor 설명의 위험

Narrative를 attractor로 표현하면 반복적 복귀를 설명하기 쉽다. 그러나 다음 현상을 지우기 쉽다.

- 장기 drift
- 불안정한 관계
- 두 상태 사이의 반복 cycle
- 새로운 상황에서의 구조 변화
- 학습에 따른 graph 자체의 수정
- 명확한 복귀점이 없는 확산

따라서 현재는 `attractor`를 필수 정의로 쓰지 않는다.

### 폐기 조건

복귀·고착·전환이 단순한 현재 상태와 정책만으로 설명되고 장기 별도 구조가 필요 없다면 Narrative-attractor 설명은 폐기한다.

---

## 7. 경로 의존성과 누락 변수

`같은 현재 상태인데 과거가 달라 미래가 다르다`는 문장은 엄밀하지 않을 수 있다.

미래가 다르면 다음 중 하나다.

```text
현재 상태가 실제로 같지 않다.
전이법이 다르다.
환경 기대가 다르다.
숨은 관계 상태가 다르다.
모델이 필요한 정보를 표현하지 못했다.
```

### 연구 원칙

경로 의존성을 별도 원리로 선언하기 전에, 현재 상태 표현에서 어떤 정보가 빠졌는지 먼저 조사한다.

---

## 8. 실제 시간의 필요성

### 비교

```text
한 시간 안에 다섯 번 위반
한 달 동안 다섯 번 위반
```

### 질문

- 사건 사이 회복이 장기 결과를 바꾸는가?
- 실제 시간만 흐르고 사건이 없을 때 감정·피로·기대가 변하는가?
- 모델이 시간 간격을 무시해도 동일한 수행을 내면 실제 시간 변수는 불필요한가?
- 시간 효과가 전부 중간 사건과 맥락 차이로 설명되는가?

---

## 9. 현재 설명의 주요 실패 조건

다음 결과가 쌓이면 현재 구조를 축소하거나 다른 흐름으로 이동해야 한다.

### 실패 A

기억 내용 하나만으로 잔여·접근 편향·장기 적응의 모든 수행 차이를 설명할 수 있다.

### 실패 B

Episode trajectory, record, residue를 나눠도 실제 예측 차이가 없다.

### 실패 C

Narrative 후보들을 선택적으로 조작할 수 없으며 모두 같은 출력을 다시 표현한 것에 불과하다.

### 실패 D

장기 경로를 명시하지 않은 직접 history 모델이 더 단순하고 더 잘 예측한다.

### 실패 E

실제 시간 간격을 바꿔도 다른 조건을 통제하면 아무 차이가 없다.

### 실패 F

경로 의존성이 모두 누락된 현재 상태 변수로 흡수된다.

---

## 10. 이 thread를 분리할 신호

- 내용 재접근과 비언어적 잔여가 독립적으로 움직인다.
- Episode 경로·기록·잔여가 별도 예측을 낸다.
- Narrative의 느린 상태와 transition 변화가 분리된다.
- 실제 시간과 사건 반복이 서로 다른 법칙을 갖는다.
- 관계적 Narrative가 개인 내부 설명으로 환원되지 않는다.

이 신호가 생기기 전에는 별도 정본 디렉터리를 만들지 않는다.
