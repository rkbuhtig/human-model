# Research Documentation Queue

이 디렉터리는 현재 작업 범위에 끼워 넣지 않을 문서 정리 문제를 보관한다.

여기에 들어온 항목은 중요하지 않아서 버린 것이 아니다. 현재 진행 중인 연구 문서의 목적을 흐리지 않도록 별도 순서로 미룬 작업이다.

## 사용 규칙

- 현재 작업 중 발견했지만 즉시 해결할 범위가 아닌 문서 문제만 둔다.
- 큐 문서는 문제, 처리 시점, 완료 조건만 기록한다.
- 큐에 넣었다는 이유로 디렉터리 구조나 문서 형식을 미리 확정하지 않는다.
- 큐 항목을 처리할 때는 해당 시점의 연구 구조와 문서를 다시 읽고 작업 범위를 정한다.
- 이 큐의 순서는 연구 주제의 중요도 순위가 아니라 문서 작업의 처리 순서다.
