# 1차 이론 내용 채움 범위

이번 작업은 PR #30에서 만든 가변 연구 흐름 구조 안에 실제 이론 내용을 처음 넣는다.

## 포함

- 다섯 thread의 현재 작업 설명
- 각 설명과 경쟁하는 대안
- 폐기·축소·분리 조건
- 다음 판별 질문
- 세 crossing의 상세 연구 내용
- 전체 내용 색인과 범위 점검

## 제외

- 상위 통합 이론
- 고정 축 승인
- Assembly
- Episode·Narrative·Self·Persona 정본화
- 프롬프트 구현
- 평가 결과
- 인간 경험적 지지 판정

## 검증할 핵심

이 문서 묶음의 가치는 설명이 매끄러운지보다 다음에 달려 있다.

```text
각 문제의 경계가 더 선명해지는가
경쟁 설명이 실제로 구분되는가
현재 개념이 틀릴 조건이 남아 있는가
새 연구 결과가 나왔을 때 구조를 다시 바꿀 수 있는가
```
