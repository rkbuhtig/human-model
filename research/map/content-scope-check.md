# 이론 내용 1차 채움 범위 점검

이 문서는 이번 내용 추가가 무엇을 채웠고 무엇을 의도적으로 남겨두었는지 확인한다.

## 추가된 내용

- 다섯 thread의 현재 작업 설명
- 각 설명과 경쟁하는 대안
- 현재 설명의 실패·폐기 조건
- 다음에 구분해야 할 질문
- 재저자화, 자원과 시간, Narrative와 사회성 교차 연구
- 전체 내용 색인

## 의도적으로 추가하지 않은 내용

- 상위 통합 이론
- Assembly 구조
- Episode·Narrative·Self·Persona 정본
- 고정 invariant registry
- formal ontology
- 프롬프트 구현
- 평가 결과
- 인간 경험적 지지 판정

## 구조 경계

이번 문서들은 현재 thread 안에 내용을 넣지만 thread 경계를 승인하지 않는다.

각 문서는 다음을 반복해서 명시한다.

- 현재 설명은 임시적이다.
- 더 단순한 경쟁 설명이 가능하다.
- 실제 수행 차이가 없으면 개념을 축소하거나 폐기한다.
- 독립적인 예측이 생기면 thread를 분리할 수 있다.
- 다른 흐름으로 환원되면 이동하거나 흡수한다.

## 다음 사용

이 문서 묶음은 즉시 하나의 완성 이론으로 합치기 위한 것이 아니다.

다음 연구에서는 각 문서에 적힌 최소 사례와 판별 질문을 사용해:

```text
경쟁 설명을 비교
→ 현재 구분이 실제 수행 차이를 만드는지 확인
→ 경계가 바뀌면 transitions/에 이유 기록
→ 살아남은 문제만 더 깊게 연구
```

한다.
