# Dormant Research

`dormant/`는 현재 적극적으로 밀지 않지만 폐기하지 않은 연구를 둔다.

잠정 중단은 다음을 뜻한다.

- 현재 mainline에서 사용하지 않음
- 현재 구조를 뒷받침하는 근거로 계산하지 않음
- 중요성 또는 가능성은 남아 있음
- 다시 열 조건을 기록함

---

## 잠정 중단 사유 예시

- 필요한 자료가 아직 없음
- 현재 프롬프트 평가로 경쟁 설명을 구분할 수 없음
- 다른 연구 흐름의 선행 정리가 필요함
- 질문은 중요하지만 현재 우선순위가 아님
- 현재 이름과 경계가 불명확해 적극 연구하기 어려움
- 이론이 아니라 장기적 직관 수준에 머물러 있음

---

## 권장 기록

```text
무엇을 연구하려 했는가
왜 아직 중요하다고 보는가
왜 지금 중단하는가
현재 무엇이 부족한가
어떤 발견이나 자료가 생기면 다시 여는가
어느 thread와 연결되어 있는가
```

`dormant/`는 불편한 문제를 숨기는 보관함이 아니다.
재개 조건 없이 무기한 밀어 넣지 않는다.
