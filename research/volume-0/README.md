## Current result of the minimal audit

The strongest directly recurrent December separation is:

```text
output / proposal / derived framing
≠ evidence
≠ commitment
≠ state write
```

It first appears as an engineering response to self-contaminating persona output. Later kernels preserve it through `COMMIT_GATE`, APPLY-only mutation, PLAN-only rendering, constraint modules without commit authority, and StoryField no-backflow.

The current program generalizes that recurrence into a candidate typed authority topology:

```text
local causal influence
≠ cross-domain certification authority
```

That generalization is current synthesis. The December engine did not already contain the complete present theory.
